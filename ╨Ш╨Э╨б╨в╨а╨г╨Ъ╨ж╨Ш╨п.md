## Деплой на Railway (пошагово)

### 1. Загрузите файлы на GitHub

Создайте репозиторий на github.com и загрузите все файлы из этой папки:
```
Dockerfile
railway.json
pb_schema.json
ИНСТРУКЦИЯ.md
frontend/
pb_hooks/
```

### 2. Подключите Railway

1. Зайдите на railway.app → New Project → Deploy from GitHub repo
2. Выберите ваш репозиторий
3. Railway найдёт Dockerfile и запустит сборку (~3 минуты)
4. Settings → Networking → Generate Domain → скопируйте URL

### 3. Пропишите ваш URL в api.js

Откройте `frontend/js/api.js`, первая строка:
```js
const PB_URL = 'https://ВАШ_ПРОЕКТ.up.railway.app';
```
Замените на ваш реальный URL, сделайте git push.

### 4. Настройте PocketBase Admin (один раз)

Откройте: https://ВАШ_ПРОЕКТ.up.railway.app/_/

- Создайте аккаунт администратора базы данных (любой email + пароль)
- Settings → Import collections → вставьте содержимое pb_schema.json → Import
- Нажмите Redeploy в Railway

### 5. Готово

Откройте https://ВАШ_ПРОЕКТ.up.railway.app

```
Логин:  admin
Пароль: 12345
```

Хуки автоматически создадут:
- 4 роли (Администратор, Мастер цеха, Магазин, Опт)
- Пользователя admin с паролем 12345

Войдите как admin → раздел «Роли и пользователи» → кнопка «+ Новый пользователь».
